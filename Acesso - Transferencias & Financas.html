Transferências — Componentes

Componentes
  • TransferBanner status
  • TabBar (Contratar / Vender / Empréstimos)
  • FilterRow (pills + selects)
  • TransferPlayerCard ×n
  • HotBadge ("🔥 Disputado")
  • ProposalModal bottom sheet
  • InputField / Select / Toggle shared
  • BottomNav shared
Estados
  • Janela aberta (exibido)
  • Janela fechada → banner cinza "Janela fechada. Volta em X dias"
  • Sem resultados de filtro → "Nenhum jogador encontrado 🔍"
  • Proposta enviada → toast "Proposta enviada! 📩 Aguarde resposta"
  • Proposta aceita → confetti verde + "Bem-vindo ao clube! 🎉"
  • Proposta recusada → toast "Proposta recusada 😔 Tente outro valor"
  • Sem saldo → botão disabled + "Sem grana 💸"
Microcopy
  • Banner → "🔥 Janela aberta! 18 dias restantes"
  • Sem clube → itálico "Sem clube" (livre, sem taxa)
  • Disputado → "🔥 Disputado" badge amarelo
  • Modal CTA → "Enviar proposta ✉️"
  • Toggle anti-rival → protege contra venda pra rival
  • Toggle revenda → "15% volta pro seu caixa"

Finanças — Componentes

Componentes
  • FinanceStatCard ×4 grid
  • FairPlayBar
  • BarChart CSS puro
  • TransactionsTable
  • ClubStructureCard ×n
  • UpgradeButton outline
  • BottomNav shared
Estados
  • Saldo positivo (exibido) — verde
  • Saldo negativo → vermelho + "No vermelho 🚨"
  • Fair play < 70% → barra verde "Dentro do fair play ✅"
  • Fair play 70–85% → barra amarela "⚠️ Atenção"
  • Fair play > 85% → barra vermelha "🚨 Risco de punição!"
  • Upgrade sem saldo → botão disabled + tooltip "R$ X necessários"
Microcopy
  • Fair play ok → "Tudo certo por aqui ✅"
  • Fair play apertado → "Cuidado com a folha! ⚠️"
  • Upgrade base → "Revela jovens melhores"
  • Construir estádio → "Construa o seu! 🏟️"
  • Transação receita → ícone + valor verde
  • Transação despesa → ícone + valor vermelho

Fluxo de Navegação

  • BottomNav "Mercado" → Transferências (aba Contratar)
  • Transferências → tap "Fazer proposta" → ProposalModal
  • ProposalModal → "Enviar" → toast + lista atualizada
  • Transferências → aba "Vender" → lista do próprio elenco
  • Transferências → aba "Empréstimos" → ofertas recebidas/enviadas
  • BottomNav "Finanças" → tela Finanças
  • Finanças → "Melhorar" → modal de confirmação de upgrade
  • Finanças → "Construir próprio" → wizard de construção

Decisões de UX

  • Modal de proposta é bottom sheet — padrão mobile nativo
  • Toggle anti-rival e revenda como opções visuais (não checkbox)
  • Banner de janela aberta no topo — urgência visual
  • Badge "Disputado" para leilão ativo — competição entre clubes
  • Jogador "Sem clube" destaca "Livre 🟢" — sem taxa de transferência
  • Fair play bar com meta visual — jogador entende o limite
  • Gráfico de barras CSS puro — não precisa de lib de charts
  • Estrutura do clube com níveis de estrela — gamificação clara
  • Transações com emoji por tipo — scan visual rápido
