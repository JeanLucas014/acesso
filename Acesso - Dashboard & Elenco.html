Dashboard — Componentes

Componentes
  • ClubHeader layout
  • StatCard ×3
  • MatchCard (VS layout)
  • ResultBadge ×5
  • AlertCard
  • BottomNav shared
Estados
  • Com dados (exibido)
  • Carregando — skeleton em cada card
  • Sem próximo jogo — card "Sem jogos agendados ⚽"
  • Sem alertas — seção oculta
  • Erro — retry com toast "Ops, tenta de novo 🔄"
Microcopy
  • Saldo negativo → "No vermelho 🚨"
  • Moral < 50% → "Vestiário pegando fogo 🔥"
  • 5 vitórias seguidas → "Tá voando! 🚀"
  • Alerta contrato → "Renove antes do pré-contrato!"

Elenco — Componentes

Componentes
  • PageHeader
  • FilterPills scroll-x
  • PlayerCard ×n
  • PositionBadge 6 cores
  • MoralBar
  • StatusBadge (Lesionado/Suspenso)
  • BottomNav shared
Estados
  • Com dados (exibido)
  • Filtro ativo — filtra lista por posição
  • Elenco vazio → "Bora contratar! 🏪"
  • Carregando — skeleton cards
  • Jogador tap → sheet com detalhes e ações
Microcopy
  • Rating ≥ 80 → "Craque! ⭐"
  • Moral < 40% → "Insatisfeito 😤"
  • Lesionado → "🤕 Lesionado — X jogos"
  • Suspenso → "🟨 Suspenso — X jogo"
  • Fadiga > 80% → "Precisa descansar 😴"

Fluxo de Navegação

  • Dashboard → tap "Próximo jogo" → Pré-jogo
  • Dashboard → tap alerta → Detalhe do jogador
  • Dashboard → BottomNav → qualquer tela
  • Elenco → tap jogador → Sheet de detalhes
  • Elenco → filtro posição → lista filtrada
  • Elenco → tap "Escalar" → Pré-jogo

Decisões de UX

  • Resultados usam V/E/D (pt-BR) — não W/D/L
  • Saldo abreviado (287,4k) nos cards compactos
  • Moral como barra visual + % numérico no tooltip
  • Fadiga invertida: 🔋 85% = 85% energia restante
  • Position badges sólidos (não outline) — legibilidade
  • Status badges no footer do card, não overlay
  • Bottom nav com 5 itens (padrão iOS/Android)
  • Escudo SVG shield — placeholder escalável
