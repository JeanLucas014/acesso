Login — Componentes

Componentes
  • LogoArea (ícone SVG + título + subtítulo)
  • InputField shared
  • PrimaryButton shared
  • TextLink
Estados
  • Default (exibido)
  • Loading — botão com spinner, inputs disabled
  • Erro de credenciais — toast "E-mail ou senha incorretos 😬"
  • Campo vazio — borda vermelha + "Preenche aí!"
  • Sucesso → redireciona para Dashboard ou CreateClub
Microcopy
  • Placeholder email → "seu@email.com"
  • Placeholder senha → "••••••••"
  • CTA → "Entrar 🟢"
  • Criar conta → link verde destacado
  • Tagline → "Da várzea à Libertadores."
Auth Flow (Supabase)
  • supabase.auth.signInWithPassword()
  • Se novo usuário sem clube → CreateClub
  • Se tem clube → Dashboard
  • Esqueci senha → supabase.auth.resetPasswordForEmail()

Criar Clube — Componentes

Componentes
  • StepProgress wizard
  • BudgetBar
  • DraftPlayerCard grid 2col
  • AttributeBar ×5
  • PositionBadge shared
  • SelectedRoster (lista compacta)
  • PrimaryButton shared
Estados
  • Draft ativo (exibido)
  • Folha > 90% → barra vermelha + alerta "Cuidado! 💸"
  • Folha 100% → botão "+" desabilitado nos cards
  • Elenco < 18 → botão "Próximo" desabilitado
  • Elenco = 22 → pool oculto + "Time completo! ✅"
Microcopy
  • Título → "Monte seu time dos sonhos (com o orçamento que você tem 😅)"
  • Folha ok → barra verde
  • Folha apertada → barra verde→amarela gradiente
  • Folha estourada → "Sem grana! Tire alguém 💸"
  • CTA etapa 4 → "Próximo: Formação →"

Fluxo de Navegação

  • Login → "Entrar" → Dashboard (se tem clube)
  • Login → "Entrar" → CreateClub (se novo)
  • Login → "Criar conta" → Signup → CreateClub
  • CreateClub Step 1 → 2 → 3 → 4 → Dashboard
  • CreateClub qualquer etapa → "Voltar" → etapa anterior
  • CreateClub etapa 4 → "Começar temporada" → Dashboard

Decisões de UX

  • Login centralizado vertical — foco total no form
  • Logo usa SVG campo de futebol estilizado (não emoji)
  • Wizard mostra etapa 3 (a mais complexa) como referência
  • Draft em grid 2col — cabe 4 cards por scroll na tela
  • Atributos em barras (não números) — mais visual e gamificado
  • Budget bar usa gradiente verde→amarelo — feedback instantâneo
  • Roster list scrollável com max-height — campo sempre visível
  • Botão "×" remove do elenco — sem confirmação (ação leve)
  • Wizard não tem "Pular" — todas as etapas são obrigatórias
