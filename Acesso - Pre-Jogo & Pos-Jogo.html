  • Rating ≤ 5.0 → vermelho "Dia pra esquecer"
  • Moral +8 → "😄 Vestiário feliz"
  • Moral -10 → "😡 Clima pesado"
  • Lesão → "🤕 Lesão leve (X jogos)"
  • CTA → "Próxima rodada →"

Campo SVG — Especificação

Dimensões
  • ViewBox: 320×440 (retrato, visão de cima)
  • Fundo: #064E3B (verde escuro)
  • Linhas: #0D7B58 (verde médio), 1.5px
  • Marcações: centro, áreas, meia-lua, escanteios
Player Circles
  • Raio: 22px
  • Fill: #22C55E (titular) / #52525B (vazio)
  • Texto: número da camisa + 3 letras do nome
  • Font: Inter 700 11px (número) + 600 8px (nome)
  • Interação: tap para trocar jogador (bottom sheet)
Formações suportadas
  • 4-3-3 (exibida) — posições pré-calculadas
  • 4-4-2 / 3-5-2 / 4-2-3-1 / 5-3-2
  • Cada formação = array de {x, y} para 10 jogadores + GK fixo

Decisões de UX

  • Campo em retrato (não paisagem) — melhor uso do espaço mobile
  • Ataque no topo do campo — convenção de jogos de futebol
  • Força do adversário sem número — evita min-maxing, mais imersivo
  • Últimos 3 resultados do adversário — info suficiente sem overload
  • Timeline vertical com dot colors — scan rápido dos eventos
  • Ratings em grid 2col — cabe todo o time sem scroll excessivo
  • Cores de rating: ≥7.0 verde, 5.5–6.9 amarelo, ≤5.4 vermelho
  • Impacto no elenco logo após ratings — causa e efeito claros
  • Botão "Próxima rodada" fecha o ciclo → Dashboard
